# 🐾 Desktop Pet 🐾

A lightweight interactive desktop pet application written in C.

The core philosophy of this project is to explore native Operating System APIs by 
building seamless desktop integrations with no frameworks. Every window, tray icon, 
and notification is driven directly by the OS's native sub-systems.

This project is a simple tool / game made in C to learn more about the different OS's API. I try to make it use the least libraries appart from the default OS's ones.
